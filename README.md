# Менеджер заявок (Учебный проект)

Простое и надежное приложение на Android для создания и просмотра заявок.
Сделано с использованием современных инструментов разработки (MAD).

### Что внутри:
* **Интерфейс:** Jetpack Compose (Material 3). Формы, списки, поиск.
* **База данных:** Room. Заявки сохраняются в локальное хранилище.
* **Сеть:** Ktor Client. Отправка данных в Google Таблицу через Apps Script.
* **Архитектура:** MVVM + Clean Architecture (Data, Domain, UI).
* **Инъекция зависимостей:** Hilt (Dagger).

### Как это работает:
1. Заполняете форму новой заявки (номер, адрес, описание).
2. При нажатии «Отправить» заявка сохраняется в базу и улетает в облако.
3. На главном экране список автоматически группируется по датам.
